What is Tableau and how does it work?

August 26, 2019August 27, 2019ShamrockBusiness Intelligence, Database Management Consulting

 

Not many companies say they have a truly modern reporting platform and regularly base decisions on the day to day ERP supplied basic reporting. In fact, most companies struggle with understanding their data and making educated decisions based on the information. When the time has come to modernize their reporting platform Tableau is one of the excellent options out in the market.

Tableau is a very powerful and fastest growing data visualization tool used in the Business Intelligence and Data Analysis Industry. Tableau turns raw data very easily into the understandable visual format.

Data analysis is very fast with Tableau and even the non-technical user can create a customized dashboard which can be utilized by individuals at any level in an organization.

How does Tableau work?

Tableau connects and extracts the data stored in numerous different platforms. From a simple source such as an excel, PDF, to a complex source like Oracle, SQL, a cloud database such as Amazon web services, Microsoft Azure SQL database, Google Cloud SQL.

When Tableau is launched, various data connectors are available which allows you to connect to different data sources.

The pulled data can either be connected live or imported to the Tableau’s data engine, Tableau Desktop. This is where the Data analyst work with the data that was pulled up and develop visualizations. The created dashboards are shared with the users as a static file. The users who receive the dashboards views the file using Tableau Online.

The data from the Tableau Desktop can be published to the Tableau Server. This is an enterprise platform where collaboration, distribution, governance, security model, automation features are supported. With the Tableau Server, the end users have a better experience in accessing the files from all locations be it a desktop, mobile or email.
